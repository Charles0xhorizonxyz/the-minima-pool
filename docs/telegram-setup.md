## Initial Settings

- Type: public group or public channel plus linked discussion group
- Public link: `t.me/theminimapool`
- History for new members: visible
- Reactions: enabled
- Slow mode: 10 seconds at launch
- New member permissions: no media/stickers/links until the group is stable
- Admins: start with your personal Telegram account only
- Bots: none at launch unless clearly needed
- Cross-promotion: none at launch

## Recommended Structure

Use a public channel for announcements and a linked public group for discussion if Telegram allows the handle setup cleanly.

If only one surface is created first, create the public group first:

```text
The Minima Pool
https://t.me/theminimapool
```

## Account Guidance

It is acceptable to create the group from the same personal Telegram user if that is your normal owner account.

Keep the group operationally dedicated:

- Use The Minima Pool name and icon
- Use The Minima Pool links only
- Start with only your personal account as admin
- Do not add unrelated project bots at launch
- Do not import old member lists
- Do not cross-link unrelated project channels in the pinned message
